**Projeto Fogão Afeto — Site de Restaurante (HTML/CSS)**

**Visão Geral**

- **Descrição:** Projeto simples de site para um restaurante desenvolvido apenas com HTML e CSS, criado para uma família que deseja lançar seu restaurante online.
- **Tema:** Livre (design acolhedor e caseiro — "Fogão Afeto").

**Conteúdo do Projeto**

- `index.html`: Página inicial com apresentação do restaurante e navegação.
- `cardapio.html`: Página com o cardápio e imagens dos pratos.
- `contato.html`: Página de contato com informações para reserva e localização.
- `style.css`: Arquivo de estilos que aplica a identidade visual do site.
- `img/`: Pasta contendo as imagens usadas no site (pratos, ambiente, logo).

**Requisitos atendidos**

- **No mínimo 3 páginas HTML conectadas:** `index.html`, `cardapio.html` e `contato.html` estão interligadas via links de navegação.
- **Estrutura em HTML e CSS:** Marcação semântica nas páginas e estilos centralizados em `style.css`.
- **Uso de imagens:** Imagens ilustrativas armazenadas em `img/` e usadas para reforçar o apelo visual.
- **História de Usuário:** Incluída abaixo.
- **Aplicação de conceitos de UX e UI:** Análise e decisões resumidas na seção de UX/UI.

**História de Usuário**

- **Como:** Pessoa responsável pela família (ex.: Ana) quer divulgar o novo restaurante para atrair vizinhos e receber pedidos e reservas.
- **Quero:** uma página inicial que conte a história do restaurante, um cardápio claro e uma forma simples de contato.
- **Para que:** clientes encontrem o cardápio facilmente, vejam imagens apetitosas dos pratos e entrem em contato para reservar ou pedir.
- **Critérios de aceitação:** conseguir navegar entre as três páginas, localizar o telefone/e-mail na página de contato e visualizar imagens do cardápio.

**Análise de UX e UI (resumo)**

- **Hierarquia visual:** títulos e seções bem definidas para guiar a leitura; uso de imagens grandes no cardápio para atrair atenção.
- **Tipografia e legibilidade:** fonte legível com contraste adequado entre texto e fundo para facilitar a leitura.
- **Navegação clara:** menu no topo presente em todas as páginas para entrada e saída rápidas entre seções.
- **Foco nas tarefas do usuário:** CTAs e informações de contato destacadas para facilitar reservas/pedidos.
- **Consistência visual:** paleta de cores e espaçamento uniformes controlados por `style.css` para sensação de marca.
- **Acessibilidade básica:** uso de elementos semânticos (`header`, `nav`, `main`, `footer`) e textos alternativos nas imagens (onde aplicável).

**Critérios de Avaliação (mapeamento)**

- **Estrutura do código HTML:** arquivos organizados com marcação semântica e links entre páginas.
- **Análise de UX e UI:** decisões descritas na seção anterior para melhorar usabilidade e estética.
- **Criatividade no design:** tema acolhedor e layout pensado para experiência familiar/caseira.
- **Criatividade na História do Usuário:** narrativa simples voltada a necessidades reais de um restaurante familiar.

**Como visualizar o site**

- Abra o arquivo `index.html` no navegador (duplo-clique ou arraste para uma aba do browser).
